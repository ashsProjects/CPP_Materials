README
======

This package includes the following files.

|-- Scheduler.java [This is the main file written in C++ which demonstrates all of the scheduliing algorithms. 
                    It has a main method which calls the three scheduler methods, which themselves call other helper methods. 
                    The list of processes are implemented as deques of deques.]
|-- README.txt [This file]

To compile:
    make all

To run:
    ./Scheduler <filename>

For example:
    ./Scheduler CS370-HW5-Input.csv
